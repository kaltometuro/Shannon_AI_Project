\# Adaptive AI Error Correction (Shannon Project)



\## Overview

This project implements an AI-based system that learns to transmit data reliably over a noisy channel.



\## Features

\- Neural encoder-decoder

\- Noise simulation

\- Bit Error Rate (BER) evaluation

\- Adaptive noise-aware encoding



\## How to Run



1\. Train the model:

python train.py



2\. Evaluate performance:

python evaluate.py



3\. Run demo:

python demo.py



\## Output

The system prints BER results across different noise levels.
